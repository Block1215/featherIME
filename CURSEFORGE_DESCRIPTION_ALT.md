[h1]FeatherにIME入力modを入れることができず、残念に思ったことはありませんか?[/h1]

[h1]✨ FeatherIME[/h1]

[quote][b][size=12px]Feather Client で IME 入力(日本語・韓国語・中国語など)を快適に使えるようにする Fabric Mod です。[/size][/b][/quote]

[h2]🔍 このModについて[/h2]

[b]FeatherIME[/b] は、Feather Client 上での IME 入力を改善するために作られた Mod です。

[url=https://modrinth.com/mod/caramel-chat]キャラメルチャット[/url] は優れた IME 改善 Mod ですが、Feather Client の内部 Mod と互換性の問題があり起動できません。
FeatherIME は Architectury を使わない純粋 Fabric 構成を採用することでこの問題を回避しています。

ネイティブライブラリは [url=https://github.com/Korea-Minecraft-Forum/CocoaInput-lib]CocoaInput-lib[/url] をベースとし、クラス設計は caramelChat のアーキテクチャを参考にしています。

[h2]✅ 機能[/h2]

[list]
[*]💬 [b]チャット・検索欄・その他 EditBox での IME 入力対応[/b]
[*]📝 [b]変換中テキストの下線表示[/b](プリエディット表示)
[*]🖥️ [b]Windows / macOS / Linux (X11・Wayland) 対応[/b](ネイティブライブラリ込み)
[*]⚡ [b]Feather Client 互換[/b] — Architectury 非依存
[/list]

[h2]⚠️ テスト状況[/h2]

[quote][b][color=#FFAA00]現在、Windows 11 + 日本語 IME (Microsoft IME) でのみ動作確認済みです。[/color][/b]
macOS・Linux・その他の言語 IME での動作は未確認です。
不具合・動作報告は [url=https://discord.gg/QZCr2NVhxD]簡易的なディスコードサーバー[/url] にお願いします。[/quote]
